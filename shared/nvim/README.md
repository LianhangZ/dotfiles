# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

```
nvim/
 - .neoconf.json
 - init.lua
 - lazy-lock.json
 - lazyvim.json
 - LICENSE
 - README.md
 - stylua.toml
 - lua/
    - config/
    - plugins/
```

- Find `/text`, for example, `/apple`. Next match `n`; Previous match `N`.
- Replace in whole file with confirm `:%s/old/new/gc`.
